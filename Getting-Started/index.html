    Getting Started

    Before using Fedoraware, make sure that you have C++ Redistributable 2019 x86 installed and updated. While Fedoraware might work on older Windows version, it is recommended to use Windows 10 to prevent any issues.

    • Download - Download the latest Fedoraware binary.
    • Injection - Inject Fedoraware.
    • Compile - Compile Fedoraware yourself (Advanced).
    • Fedoraloader - Automatically download & inject Fedoraware.
